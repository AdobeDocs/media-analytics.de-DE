# Medienübersicht{#media-overview}

Das Dashboard „Medienübersicht“ ermöglicht die Überwachung von Medien auf Ihrer gesamten Site. Die Anzeige Medienübersicht zeigt mehrere aggregierte Messungen an, sodass Sie schnell überwachen können, dass Medien die erwartete Leistung zeigen. Ein Diagramm zeigt [[!UICONTROL Inhaltsstarts]](/help/reporting/metrics/content-starts.md) [[!UICONTROL  neben „Anzeigenstarts]](/help/reporting/metrics/ad-starts.md) an, damit Sie diese Metriken für jedes Medienelement schnell anzeigen können.
