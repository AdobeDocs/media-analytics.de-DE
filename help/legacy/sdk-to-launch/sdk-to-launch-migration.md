# Migration vom Standalone Media SDK zu Adobe Launch

>[!NOTE]
>Adobe Experience Platform Launch wurde umbenannt und umfasst eine Suite von Datenerfassungstechnologien in Experience Platform. Infolgedessen wurden in der gesamten Produktdokumentation mehrere terminologische Änderungen vorgenommen. Eine Übersicht der terminologischen Änderungen finden Sie im folgenden [Dokument](https://experienceleague.adobe.com/docs/experience-platform/tags/term-updates.html?lang=de).

Diese Dokumentation konzentriert sich hauptsächlich auf die Unterschiede bei der Implementierung von Media Analytics
zwischen der eigenständigen Media SDK und der Media Analytics Launch-Erweiterung.

Launch bietet Kunden eine einfache Möglichkeit, alle Analysen bereitzustellen und zu verwalten,
Marketing- und Werbelösungen, die zur Unterstützung relevanter Kunden erforderlich sind
Erlebnisse im Web und auf Mobilgeräten. Launch ermöglicht jedem das Erstellen und Verwalten
ihre eigenen Integrationen mit Launch. Diese Integrationen werden Erweiterungen genannt.
Erweiterungen stehen Launch-Kunden in einer App-Store-Oberfläche zur Verfügung, sodass sie
können ihre Lösungen schnell installieren, konfigurieren und bereitstellen.

Adobe Media Analytics für Audio und Video ist als Launch-Erweiterung für die Bereiche Web und Mobile verfügbar:

* [Web Launch-Dokumentation](https://experienceleague.adobe.com/docs/experience-platform/tags/extensions/adobe/media-analytics/overview.html?lang=de)
* [Dokumentation zu Mobile Launch](https://developer.adobe.com/client-sdks/documentation/adobe-media-analytics/)
